# PortfolioHUB

Portfólio digital desenvolvido para apresentar minhas habilidades, projetos e trajetória acadêmica na área de tecnologia.

## Acesse o portfólio
> https://matheus-dabella.github.io/portifolioHUB/

## Sobre
Este portfólio foi desenvolvido utilizando HTML e CSS, com foco em organização de conteúdo, clareza visual e experiência do usuário.

## Projetos incluídos

### GasWay (Protótipo de App)
Protótipo acadêmico focado em UX e interface, simulando um sistema de visualização de preços e qualidade de combustíveis.

### Campo Minado em C
Implementação do jogo Campo Minado utilizando linguagem C, com foco em lógica de programação e estruturas de controle.

## Objetivo
Consolidar conhecimentos em desenvolvimento e criar uma base para evolução na área de tecnologia.
